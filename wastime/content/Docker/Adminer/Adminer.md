## Запуск Adminer для управления БД

Перед созданием проекта убедитесь, что порт `8084` не занят другим приложением!

В Windows Powershell:

```powershell
docker run -d `
  --name adminer `
  -p 8084:8080 `
  adminer:latest
```

В Git-Bash/Linux/WSL 2.0/Mac:

```bash
docker run -d \
  --name adminer \
  -p 8084:8080 \
  adminer:latest
```
